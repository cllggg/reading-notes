## 核心策略详解 (Detailed Strategies)

### 1. `tit_for_tat` (以牙还牙)

*   **中文昵称**：一报还一报 / 互惠利他
*   **六书溯源**：#合作的进化 #自私的基因
*   **原理推导**：
    *   **数学视角**：在无限次重复博弈中，设贴现因子为 $\delta$ (0 < $\delta$ < 1)。若对手背叛，TFT 立即背叛；若对手合作，TFT 合作。
        *   合作收益流：$R + R\delta + R\delta^2 + ... = \frac{R}{1-\delta}$
        *   背叛一次的收益流（假设对方下一轮报复）：$T + P\delta + P\delta^2 + ... = T + \frac{P\delta}{1-\delta}$
        *   只要 $\delta \geq \frac{T-R}{T-P}$，合作即为纳什均衡。
    *   **行为实验**：阿克塞尔罗德锦标赛冠军。它具备四个特征：善良（Nice）、可激怒（Provocable）、宽容（Forgiving）、清晰（Clear）。

*   **应用模板**：
    *   **情境**：初创公司与供应商建立长期关系。
    *   **角色**：采购经理 (Buyer) vs 供应商 (Supplier)。
    *   **行动**：首次订单全额付款（合作）；若供应商延期或质量差（背叛），立即停止下个订单或扣款（惩罚）；一旦供应商整改（合作），立即恢复正常采购（宽容）。
    *   **收益**：建立长期稳定的供应链，避免被“杀熟”。

*   **故事脚本**：
    > **[场景：SpaceX 供应链会议]**
    > **Musk**: "听着，对于那些试图在猛禽发动机零件上涨价的供应商，我们不仅要拒绝，还要在推特上公开羞辱他们（Defect）。但如果他们下个季度把价格降回来并保证质量，我们就给他们双倍订单（Forgive）。不要记仇，我们要的是火星，不是复仇。"
    > **Huang**: "同意。在芯片制造里也是一样，台积电知道如果我们背叛他们转单三星，成本会很高，但如果他们良率不达标，我们也会毫不犹豫地切分订单。这是动态的平衡。"

*   **三矩阵**：
    *   **收益**：长期合作收益 > 短期背叛收益。
    *   **风险**：遇到“完全背叛者”时会损失第一轮收益。
    *   **伦理**：符合“正义”观念，但不符合“无条件宽恕”的宗教伦理。

*   **升维降维**：
    *   **升维**：引入“声誉系统”，让背叛者的信息在网络中广播。
    *   **降维**：编写自动执行的 Smart Contract，一旦 API 检测到延迟，自动扣除质押币。

### 2. `chicken_game` (胆小鬼博弈)

*   **中文昵称**：边缘政策 / 懦夫博弈
*   **六书溯源**：#冲突的战略 #策略思维
*   **原理推导**：
    *   **数学视角**：两个司机相向而行。如果都直行（Straight），相撞（-100, -100）；如果一方转弯（Swerve），另一方直行，直行者赢（10, -1）；都转弯（0, 0）。
        *   没有纯策略纳什均衡，只有混合策略均衡。
        *   关键在于**承诺的可信度**。
    *   **行为视角**：为了赢，你必须让自己看起来“非理性”或者“没有退路”。

*   **应用模板**：
    *   **情境**：美苏核威慑 / 创始人争夺控制权。
    *   **角色**：激进派 (Hawk) vs 温和派 (Dove)。
    *   **行动**：公开宣称绝不退让，甚至扔掉方向盘（限制自己的选择权）。
    *   **收益**：迫使对方退让，获得最大利益。

*   **故事脚本**：
    > **[场景：DAO 治理攻击]**
    > **Geek**: "我们发起了 51% 攻击提案，如果社区不通过我们的分红方案，我们就销毁国库里所有的 Token。我们已经锁死了智能合约，只有通过提案才能解锁，钥匙已经扔了（Commitment）。"
    > **Sociologist**: "这是典型的边缘政策。他们在赌我们不敢看着社区归零。理性的做法是妥协，但如果我们也表现出‘宁为玉碎’的态度呢？"
    > **Musk**: "那就看谁更疯狂了。通常，最有钱的人最怕死，除了我。"

*   **三矩阵**：
    *   **收益**：赢家通吃，输家从属。
    *   **风险**：极高（双输即毁灭）。
    *   **伦理**：通常被视为不道德的勒索，但在国际政治中常见。

*   **升维降维**：
    *   **升维**：引入第三方担保或保险，降低毁灭的代价。
    *   **降维**：将“相撞”转化为“小规模冲突”，降低单次博弈的烈度。

### 3. `signaling` (信号博弈)

*   **中文昵称**：发送信号 / 昂贵信号
*   **六书溯源**：#策略思维 #自私的基因 #博弈论
*   **原理推导**：
    *   **数学视角**：信息不对称下，拥有私人信息的一方（Sender）通过行动向未拥有信息的一方（Receiver）传递信息。
        *   只有**成本高昂**的信号才是可信的（Separating Equilibrium）。
        *   若成本太低，则是混同均衡（Pooling Equilibrium），信号失效。
    *   **行为视角**：孔雀开屏（冒着被天敌发现的风险）证明自己强壮；名校学位（不仅是学知识，更是筛选智力和毅力的高成本门槛）。

*   **应用模板**：
    *   **情境**：VC 融资。
    *   **角色**：创业者 (Entrepreneur) vs 投资人 (VC)。
    *   **行动**：创业者投入全部身家（All-in），甚至卖房创业。
    *   **收益**：获得 VC 信任，拿到高估值融资。

*   **故事脚本**：
    > **[场景：英伟达发布会]**
    > **Huang**: "为什么我要穿这件皮衣？哪怕在加州的夏天？这是一个信号。它代表着一致性、坚韧和一种‘极客摇滚’的态度。更重要的是，我们每年投入数十亿研发 CUDA 生态，这是竞争对手无法模仿的‘昂贵信号’。"
    > **Sociologist**: "就像原始部落的纹身，疼痛证明了忠诚。"

*   **三矩阵**：
    *   **收益**：消除信息不对称，达成交易。
    *   **风险**：发送了昂贵信号但对方没接收到（如媚眼抛给瞎子看）。
    *   **伦理**：可能导致内卷（过度教育、奢侈品消费）。

*   **升维降维**：
    *   **升维**：建立基于区块链的不可篡改信誉记录（Soulbound Tokens）。
    *   **降维**：用简单的押金（Deposit）代替复杂的尽职调查。

### 4. `stag_hunt` (猎鹿博弈)

*   **中文昵称**：安全博弈 / 协调博弈
*   **六书溯源**：#策略思维 #博弈论
*   **原理推导**：
    *   **数学视角**：两人合作猎鹿（5, 5），单独猎兔（2, 2），一人猎鹿一人猎兔（0, 2）。
        *   两个纳什均衡：（鹿，鹿）和（兔，兔）。
        *   （鹿，鹿）是帕累托最优（Payoff Dominant），（兔，兔）是风险上策（Risk Dominant）。
    *   **行为视角**：信任不仅仅是相信对方好，而是相信对方相信我也相信他（高阶信念）。

*   **应用模板**：
    *   **情境**：开源社区维护 / 气候变化协议。
    *   **角色**：国家 A vs 国家 B / 开发者 A vs 开发者 B。
    *   **行动**：共同投入资源开发核心架构，而不是各自造轮子。
    *   **收益**：获得巨大的生态红利。

*   **故事脚本**：
    > **[场景：OpenAI 与 微软 早期合作]**
    > **Musk**: "我们要造 AGI（鹿），这很难，如果失败了我们什么都没有。如果我们去做简单的 App（兔），能赚钱但没意义。Sam，你敢不敢跟我一起去猎鹿？"
    > **Geek**: "这取决于我们对彼此‘疯狂程度’的判断。如果我们都确信对方是理想主义者，猎鹿就是唯一的选择。"

*   **三矩阵**：
    *   **收益**：合作收益极大。
    *   **风险**：对方背叛（去抓兔子）导致自己一无所获。
    *   **伦理**：强调集体主义和互信。

*   **升维降维**：
    *   **升维**：通过文化认同（Culture）建立默认的协调点（Focal Point）。
    *   **降维**：将大目标拆解为里程碑，每一步都是小的猎鹿博弈。

### 5. `mechanism_design` (机制设计)

*   **中文昵称**：逆向博弈 / 制度设计
*   **六书溯源**：#博弈论 #博弈论与经济行为
*   **原理推导**：
    *   **数学视角**：给定社会目标函数（Social Choice Function），设计游戏规则（Game Form），使得在均衡状态下，参与者的自利行为导致社会目标的实现。
        *   核心原理：激励相容（Incentive Compatibility）和显示原理（Revelation Principle）。
    *   **行为视角**：与其教导人们向善，不如设计一套规则，让坏人为了自己的利益也不得不做好事。

*   **应用模板**：
    *   **情境**：碳排放交易 / 频谱拍卖。
    *   **角色**：政府 (Designer) vs 企业 (Agents)。
    *   **行动**：设计维克里拍卖（Vickrey Auction），让每个人按真实估价报价是最优策略。
    *   **收益**：资源分配效率最大化。

*   **故事脚本**：
    > **[场景：火星殖民地宪法起草]**
    > **Sociologist**: "我们需要法律规定每个人必须工作。"
    > **Musk**: "不，法律没用。我们要设计一套机制：氧气配额与工作贡献挂钩，但允许自由交易。如果你想偷懒，你就得花高价买氧气。让‘生存欲望’成为最好的监工（Incentive Compatibility）。"
    > **Geek**: "这就像比特币的挖矿算法，没有人指挥，但算力保护了网络安全。"

*   **三矩阵**：
    *   **收益**：系统整体效率最优。
    *   **风险**：设计漏洞（Bug）被利用导致系统崩溃（如眼镜蛇效应）。
    *   **伦理**：可能导致弱者被系统性淘汰，需要“保留效用”约束。

*   **升维降维**：
    *   **升维**：利用 AI 模拟千万次博弈，寻找最优规则参数（AI Economist）。
    *   **降维**：简化规则，如“切蛋糕者后选”。
