# 《品格之路》知识库

本项目旨在系统梳理大卫·布鲁克斯（David Brooks）所著《品格之路》（The Road to Character）一书的核心内容。通过深度研读与拆解，帮助读者从“简历美德”走向“悼词美德”。

## 📚 知识库导航

### 01 核心理论 (Core Theory)
> 书籍的灵魂与框架，理解全书的基石。
- [**01_亚当一世与亚当二世**](01_章节知识要点/01_亚当一世与亚当二世.md)：理解两种人生的根本冲突与平衡。
- [**10_谦卑守则**](01_章节知识要点/10_谦卑守则.md)：全书总结出的15条道德建设准则。

### 02 案例解剖 (Case Studies)
> 伟大的灵魂是如何炼成的？针对书中核心人物的深度分析。
- [**01_弗朗西斯·帕金斯**](02_案例解剖/01_弗朗西斯·帕金斯.md)：**被召唤的人生** —— 如何将痛苦转化为使命。
- [**02_艾森豪威尔**](02_案例解剖/02_艾森豪威尔.md)：**自我征服** —— 如何通过克制与制度驯服内心的猛兽。
- [**03_多萝西·戴**](02_案例解剖/03_多萝西·戴.md)：**苦难与爱** —— 从混乱到神圣的救赎之路。
- [**04_乔治·马歇尔**](02_案例解剖/04_乔治·马歇尔.md)：**克制与制度** —— 制度人如何通过无私获得尊严。
- [**05_菲利普·伦道夫**](02_案例解剖/06_菲利普·伦道夫.md)：**尊严与抗争** —— 用极度的文明对抗野蛮。
- [**06_乔治·艾略特**](02_案例解剖/07_乔治·艾略特.md)：**爱与责任** —— 只有深度的承诺才能整合破碎的灵魂。
- [**07_奥古斯丁**](02_案例解剖/08_奥古斯丁.md)：**有序的爱** —— 重新排列爱的优先级，从欲望走向恩典。
- [**08_塞缪尔·约翰逊**](02_案例解剖/09_塞缪尔·约翰逊.md)：**自我审视** —— 与自我欺骗和懒惰的终身搏斗。

### 03 深刻洞察 (Deep Insights)
> 结合现实生活的反思与批判。
- [**01_简历美德与悼词美德**](03_深刻洞察/01_简历美德与悼词美德.md)：对现代成功学的深度解构。
- [**02_当今时代品格的意义与精英品格修炼**](03_深刻洞察/02_当今时代品格的意义与精英品格修炼.md)：深度回答“品格为什么重要”以及“如何修炼”。
- [**03_品格关键词深度解析**](03_深刻洞察/03_品格关键词深度解析.md)：对谦卑、自律、自由等12个关键词的深度哲学拆解。

### 04 知识内化 (Action Plan)
> 从“知道”到“做到”。
- [**01_个人行动指南**](04_知识内化/01_个人行动指南.md)：场景化的日常践行策略与习惯清单。

## 🛠 使用说明
建议按照 **理论 -> 案例 -> 反思 -> 行动** 的顺序进行阅读和实践。
- 阅读案例时，请对照自身，思考：“我身上是否有类似的弱点？我可以从他那里借用什么‘脚手架’？”
- 使用 [模板.md](模板.md) 来记录你自己的读书笔记。
