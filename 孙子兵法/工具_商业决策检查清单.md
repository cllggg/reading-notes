# 孙子兵法：高维商业决策量化自评表 (庙算 2.0)

## 一、自评逻辑
基于“五事七计”设计的量化自评系统。每个维度分为 0-10 分。
*   **总分 < 60**：不可战（不可胜在己，不可行）。
*   **60-80 分**：慎战（守则不足，需弥补短板）。
*   **80 分以上**：可战（胜可知，而不可为）。

---

## 二、维度评分 (Scoring)

### 1. 道 (Alignment & Mission) —— 内部凝聚力
*   **定义**：全员是否“上下同欲”？
*   **评分标准**：
    *   [ ] 核心团队目标高度一致 (1-2)
    *   [ ] 员工利益与项目成败挂钩 (1-2)
    *   [ ] 企业文化支持该项目的执行方式 (1-2)
    *   [ ] 愿景对客户有真实的吸引力 (1-2)
    *   [ ] 内部沟通透明，无明显内耗 (1-2)
*   **得分：____ / 10**

### 2. 天 (Timing & Macro) —— 外部趋势
*   **定义**：时机是否站在你这一边？
*   **评分标准**：
    *   [ ] 政策环境支持 (1-2)
    *   [ ] 处于行业上升周期 (1-2)
    *   [ ] 技术成熟度正好处于商业化爆发点 (1-2)
    *   [ ] 季节性/周期性风险可控 (1-2)
    *   [ ] 竞争对手尚未全面反应 (1-2)
*   **得分：____ / 10**

### 3. 地 (Market & Niche) —— 空间优势
*   **定义**：是否选择了有利的“地形”？
*   **评分标准**：
    *   [ ] 市场规模足够大，且增速快 (1-2)
    *   [ ] 具备天然的进入壁垒 (Moat) (1-2)
    *   [ ] 获客成本 (CAC) 远低于生命周期价值 (LTV) (1-2)
    *   [ ] 供应链/渠道处于我方掌控中 (1-2)
    *   [ ] 即使失败，退出成本也较低 (1-2)
*   **得分：____ / 10**

### 4. 将 (Leadership & Talent) —— 执行中枢
*   **定义**：团队是否有“智信仁勇严”？
*   **评分标准**：
    *   [ ] 负责人有相关成功经验 (智) (1-2)
    *   [ ] 激励机制明确且可信 (信) (1-2)
    *   [ ] 团队关怀与员工忠诚度高 (仁) (1-2)
    *   [ ] 领导者敢于承担风险与决策 (勇) (1-2)
    *   [ ] 管理流程严密，无执行死角 (严) (1-2)
*   **得分：____ / 10**

### 5. 法 (System & Resource) —— 资源保障
*   **定义**：组织架构与资源是否匹配？
*   **评分标准**：
    *   [ ] 预算充足且现金流稳健 (1-2)
    *   [ ] 组织架构扁平，决策速度快 (1-2)
    *   [ ] 核心流程已标准化 (SOP) (1-2)
    *   [ ] 关键设备/技术自主可控 (1-2)
    *   [ ] 风险预警与止损机制完善 (1-2)
*   **得分：____ / 10**

7.  **赏罚孰明**：谁的考核制度更公平？ ( )
*   **七计总分：____ / 70**

---

## 四、最终决策建议 (Conclusion)

*   **五事得分 (50) + 七计得分 (70) = 总分 (120)**
*   **决策 SOP**：
    1.  **分值 < 70**：**立刻停止**。目前属于“无算”，开战必殆。
    2.  **70 < 分值 < 90**：**慎重进入**。识别评分最低的 3 个子项，进行专项补强（先为不可胜）。
    3.  **分值 > 90**：**全速推进**。此时处于“胜兵先胜”状态，应集中资源发起“饱和攻击”。

---
**使用说明**：
由决策委员会 3-5 人独立评分后取平均值。重点讨论分歧最大的子项（信息不对称点）。
