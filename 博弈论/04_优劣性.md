# 04_优劣性 (Matrices: Payoff, Risk, Ethics)

> **"Every decision has a price. The question is: who pays?"**
> — *The Hacker (Geek Persona)*

本章节对核心策略进行三维评估：收益（Payoff）、风险（Risk）与伦理（Ethics）。

## 1. 以牙还牙 (`tit_for_tat`)

### 矩阵一：收益矩阵 (Payoff Matrix)
*假设情境：长期商业合作*
| 策略组合 | 长期收益 (NPV) | 市场份额 | 品牌资产 |
| :--- | :--- | :--- | :--- |
| **双方合作** | High (双赢) | 稳定增长 | 提升 |
| **我合作，对方背叛** | Low (第一轮亏损) | 损失 | 不变 |
| **我背叛，对方合作** | Medium (短期暴利，长期归零) | 短期激增，随后崩溃 | 受损 |
| **双方背叛** | Very Low (双输) | 萎缩 | 恶化 |

### 矩阵二：风险矩阵 (Risk Matrix)
| 风险类型 | 描述 | 应对措施 |
| :--- | :--- | :--- |
| **误判风险** | 对方其实想合作，但因操作失误被我判定为背叛，导致冤冤相报。 | 引入“宽容度” (Generous TFT)，允许 10% 的误操作。 |
| **被剥削风险** | 遇到“总是背叛”的对手，我会损失第一轮成本。 | 设置止损点，一旦确认对手是“恶人”，立即切换为“总是背叛”。 |

### 矩阵三：伦理矩阵 (Ethics Matrix)
| 维度 | 评分 (1-10) | 评价 |
| :--- | :--- | :--- |
| **正义性** | 9 | “以直报怨”，符合大多数人的道德直觉。 |
| **利他性** | 7 | 虽然是为了自利，但在客观上促进了合作。 |
| **残酷性** | 3 | 有报复机制，但目的是惩罚而非毁灭。 |

## 2. 胆小鬼博弈 (`chicken_game`) / 边缘政策

### 矩阵一：收益矩阵 (Payoff Matrix)
*假设情境：核威慑 / 融资对赌*
| 策略组合 | 我方收益 | 对方收益 | 备注 |
| :--- | :--- | :--- | :--- |
| **我进，你退** | +100 (全胜) | -10 (屈辱) | 最佳结果 |
| **我退，你进** | -10 (屈辱) | +100 (全胜) | 最差结果（除了死） |
| **双方退** | 0 (平局) | 0 (平局) | 维持现状 |
| **双方进** | -1000 (毁灭) | -1000 (毁灭) | 熔断点 |

### 矩阵二：风险矩阵 (Risk Matrix)
| 风险类型 | 描述 | 应对措施 |
| :--- | :--- | :--- |
| **误判风险** | 对方真的疯了，或者对方没看到我的刹车坏了。 | 建立“热线电话”，确保信息传达准确；不要把对手逼到死角。 |
| **失控风险** | 局势升级速度超过了决策速度。 | 设置自动熔断机制（Fail-safe）。 |

### 矩阵三：伦理矩阵 (Ethics Matrix)
| 维度 | 评分 (1-10) | 评价 |
| :--- | :--- | :--- |
| **正义性** | 2 | 这种策略本质上是勒索。 |
| **利他性** | 0 | 完全自利。 |
| **残酷性** | 10 | 拿生存作为赌注。 |

## 3. 猎鹿博弈 (`stag_hunt`) / 协调博弈

### 矩阵一：收益矩阵 (Payoff Matrix)
*假设情境：生态系统共建*
| 策略组合 | 我方收益 | 对方收益 | 备注 |
| :--- | :--- | :--- | :--- |
| **都猎鹿** | +50 (大餐) | +50 (大餐) | 帕累托最优 |
| **都猎兔** | +5 (小吃) | +5 (小吃) | 风险上策 |
| **我鹿，你兔** | 0 (饿死) | +5 (小吃) | 信任错付 |
| **我兔，你鹿** | +5 (小吃) | 0 (饿死) | 背叛获利极小 |

### 矩阵二：风险矩阵 (Risk Matrix)
| 风险类型 | 描述 | 应对措施 |
| :--- | :--- | :--- |
| **协调失败风险** | 只要有一丝怀疑对方会去抓兔子，理性的我也只能去抓兔子。 | 建立高频沟通机制；签订履约保函。 |
| **搭便车风险** | 对方在猎鹿过程中出力少。 | 设计贡献度衡量系统。 |

### 矩阵三：伦理矩阵 (Ethics Matrix)

## 4. 机制设计 (`mechanism_design`) / 维克里拍卖

### 矩阵一：收益矩阵 (Payoff Matrix)
*假设情境：公共资源拍卖*
| 策略 | 真实报价 ($v$) | 虚假报价 ($b \neq v$) |
| :--- | :--- | :--- |
| **赢得拍卖** | 支付第二高价，剩余收益 $v - p_{2nd}$ | 可能支付过高 ($b > v$) 导致亏损，或支付过低 ($b < v$) 错失机会 |
| **输掉拍卖** | 0 | 0 |

### 矩阵二：风险矩阵 (Risk Matrix)
| 风险类型 | 描述 | 应对措施 |
| :--- | :--- | :--- |
| **合谋风险** | 竞标者之间串通压价。 | 引入保留价 (Reserve Price)；对合谋行为进行重罚。 |
| **隐私风险** | 真实报价暴露了商业机密。 | 使用零知识证明 (ZKP) 或加密拍卖。 |

### 矩阵三：伦理矩阵 (Ethics Matrix)
| 维度 | 评分 (1-10) | 评价 |
| :--- | :--- | :--- |
| **正义性** | 10 | 理论上最公平的机制，资源流向最需要它的人。 |
| **利他性** | 5 | 设计者的目的是社会总福利最大化。 |
| **残酷性** | 4 | 赢家通吃，输家一无所有。 |
