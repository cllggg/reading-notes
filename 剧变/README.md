# 《剧变》深度研读笔记：个人与国家危机的转折点

> **“危机不仅是危险，更是选择性变革的机会。” —— 贾雷德·戴蒙德**

---

## 💎 炼金导读

在《枪炮、病菌与钢铁》探究了地理的终极因，《崩溃》分析了社会如何毁灭后，戴蒙德在《剧变》中转向了更具“主体性”的命题：**当危机降临时，为什么有些国家/个人能够涅槃重生，而有些却从此沉沦？**

这本书建立了一个极具启发性的桥梁：将**个人危机疗法**的 12 个维度迁移到**国家治理**中。它告诉我们，应对剧变的本质不在于全盘否定过去，而在于**“选择性变革”**。

## 🌟 本书核心金句

*   “我们无法改变地理，但我们可以改变地理所意味的一切。”
*   “成功应对危机的关键在于愿意承担责任，而不是将自己视为受害者。”
*   “选择性变革：在改变某些事物的同时，保留那些最珍贵、最独特的部分。”
