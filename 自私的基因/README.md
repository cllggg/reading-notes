# 《自私的基因》读书笔记

> 作者：理查德·道金斯 (Richard Dawkins)

## 简介
《自私的基因》是进化生物学领域最具影响力的著作之一。道金斯在书中提出，进化的基本单位不是物种，也不是群体，而是基因。

## 🎯 阅读指南

### 初学者建议
1. 先阅读[核心概念索引](./00_核心概念索引.md)了解关键术语
2. 按顺序阅读第1-4章，建立理论基础
3. 重点阅读第5章（ESS）和第10章（互惠利他）
4. 完成[常见问题解答](./00_常见问题.md)中的思考题

### 深度学习者建议
1. 精读全书16章
2. 完成每章的思考题和实践练习
3. 阅读[延伸阅读推荐](./00_延伸阅读.md)中的相关书籍
4. 尝试将理论应用到实际生活

### 实践应用者建议
1. 重点阅读第4、5、10、12章
2. 完成每章的实践练习
3. 应用ESS、TFT等策略到工作和生活中
4. 建立"基因行为日记"，持续观察和优化

## 💡 核心洞察

1. **基因是进化的基本单位**，个体只是基因的"生存机器"
2. **"自私"是功能性术语**，与主观动机无关
3. **利他行为在基因层面仍然是"自私"的**（亲缘选择+互惠利他）
4. **人类是唯一能反抗基因暴政的物种**
5. **看清了牢笼的形状，才是自由的开始**
