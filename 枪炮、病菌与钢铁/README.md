# 《枪炮、病菌与钢铁》：人类社会的命运炼金术

> **核心命题**：不同民族间历史命运的差异，不在于生物学上的优劣，而在于环境的差异。

---

## 🧭 笔记地图

### 1. [终极因：地理环境的底层算法](01_终极因：地理环境的底层算法.md)
*   **关键词**：地理决定论、可驯化动植物、大陆轴线。
*   **炼金点**：为什么是“地理”决定了财富与权力的分配？

### 2. [直接因：枪炮、病菌与钢铁的暴力美学](02_直接因：枪炮、病菌与钢铁的暴力美学.md)
*   **关键词**：技术溢出、免疫力、组织形态。
*   **炼金点**：从环境中长出来的“武器”是如何改变世界的。

### 3. [大陆轴线：为什么是欧亚大陆的胜利？](03_大陆轴线：为什么是欧亚大陆的胜利？.md)
*   **关键词**：东西向 vs 南北向、气候屏障、传播效率。
*   **炼金点**：地理方向如何决定了文明传播的速度。

### 4. [病菌的契约：驯化与致命的馈赠](04_病菌的契约：驯化与致命的馈赠.md)
*   **关键词**：人畜共患病、免疫护城河、生物殖民。
*   **炼金点**：看不见的杀手如何重塑世界版图。

### 5. [跨界交融：地理决定论遇见现代商业与地缘政治](05_跨界交融：地理决定论遇见现代商业与地缘政治.md)
*   **关键词**：路径依赖、资源禀赋、第二曲线。
*   **炼金点**：用戴蒙德的视角看现代大国的博弈与企业的兴衰。

### 6. [从历史中炼金：给现代人的生存启发](06_从历史中炼金：给现代人的生存启发.md)

*   **炼金点**：当环境已经注定，我们如何通过“认知炼金”改写个人算法？

---

## 💎 炼金术士的视角

这本书不仅仅是在讲历史，它提供了一套**“环境-资源-技术-组织-权力”**的宏大演化模型。学习这本书，是为了让我们在面对当下的困境时，能够跳出“个人努力”的迷思，看到那些潜伏在冰山之下的、长周期的决定性力量。

---
*“不同民族间历史命运的差异，是因为他们所处的环境不同，而不是因为他们本身在生物学上有什么差异。” —— 贾雷德·戴蒙德*
