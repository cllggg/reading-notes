# 个人行动指南 (Action Plan)

## 1. 🎯 核心目标
将《品格之路》中的“悼词美德”转化为日常生活中的微习惯和决策准则。不仅要“知道”，更要“做到”。

## 2. 🗓️ 每日/每周 习惯清单 (Habits)

### 每日践行
- [ ] **晨间反思 (5分钟)**：
    - 今天的日程中，哪件事是“亚当一世”（简历）的，哪件事是“亚当二世”（品格）的？
    - 今天我将如何服务他人？
- [ ] **情绪缓冲区**：
    - 遇到愤怒或冲动时，暂停 10 秒（艾森豪威尔法则）。
    - 提醒自己：“这不是关于我，而是关于大局。”
- [ ] **微小的善行**：
    - 做一件不求回报、不为人知的小事（捡起垃圾、给陌生人开门、真心赞美同事）。

### 每周复盘（安息日）
- [ ] **独处一小时**：
    - 远离手机和网络。
    - 阅读经典或精神类书籍（如《沉思录》、《道德经》）。
- [ ] **道德账单**：
    - 回顾本周的言行，有哪些是傲慢的？有哪些是虚荣的？
    - 记录一个具体的改进点。

## 3. 🛠️ 场景化应对策略 (Scenarios)

### 场景一：当面临职业选择时
- **原则**：不要只问“这能给我带来多少钱/名声？”，要问“这会让我成为什么样的人？”、“这是否符合我的神圣召唤？”
- **行动**：列出选项，对比其对“简历”和“悼词”的影响。优先选择那些能磨练技能、服务社会、且需要克服困难的选项（帕金斯法则）。

### 场景二：当遭遇失败或批评时
- **原则**：痛苦是成长的催化剂。不要防御，要接纳。
- **行动**：
    1.  承认痛苦，不逃避。
    2.  问自己：“这个失败揭示了我性格中的什么弱点？”
    3.  制定针对该弱点的修正计划（如：如果是因傲慢而败，就练习倾听）。

### 场景三：当取得成功或受到赞誉时
- **原则**：警惕傲慢。成功是副产品，不是终点。
- **行动**：
    1.  将功劳归于团队、运气和外部支持（马歇尔法则）。
    2.  立即设定下一个更具挑战性、更有服务性质的目标，防止停留在功劳簿上。

## 4. 📝 自我誓言 (The Vow)
> 参考书中的“谦卑守则”，写下属于自己的誓言。

1. 我承认我有缺陷，我并不完美，我需要不断的自我修正。
2. 我承诺通过工作来服务他人，而不仅仅是满足私欲。
3. 我将在混乱中建立秩序，在冲动中保持克制。
4. 我相信爱是动词，是责任，是牺牲。
5. 我将终身致力于缩小“现在的我”和“应该成为的我”之间的差距。
