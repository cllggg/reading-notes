## 🛠️ Tool 1: 影响力自检表 (Self-Checklist)

> **说明**：请花5分钟，诚实回答以下问题。每项1-5分（1=从不，5=总是）。
> **得分**：
> *   **<20分**：隐形人。你需要大声说出你的需求。
> *   **20-40分**：普通人。你的影响力主要靠运气。
> *   **>40分**：影响者。你懂得如何调动人心。

### 维度 A：情感连接 (Heart)
1.  **情绪感知**：当别人生气时，我能准确叫出他的情绪名字（愤怒/委屈/焦虑），而不是急着辩解。 [ ]
2.  **互惠预埋**：在求人办事之前，我会先主动提供一个小小的帮助或关怀。 [ ]
3.  **喜好建立**：我能在陌生的社交场合迅速找到与对方的共同点（老乡/校友/爱好）。 [ ]

### 维度 B：逻辑框架 (Head)
4.  **价值重构**：当遇到反对意见时，我能把对方的“攻击”重新解释为“需求”。 [ ]
5.  **承诺引导**：我善于让对方先答应一个小请求，从而锁死后续的大承诺。 [ ]
6.  **故事包装**：我能把枯燥的数据或道理，编成一个有角色、有冲突的故事。 [ ]

### 维度 C：权威与场域 (Hand)
7.  **社会认同**：我会引用“大多数人”或“类似案例”来证明我的观点。 [ ]
8.  **权威背书**：在关键时刻，我会引用专家数据或行业标准来增加分量。 [ ]
9.  **稀缺营造**：我会在谈判中适时透露“机会有限”或“竞争激烈”的信息。 [ ]
10. **预期管理**：我会提前告知风险（丑话说前头），从而降低对方的心理预期。 [ ]

## 🛠️ Tool 2: 说服力脚本生成器 (Script Generator)

> **场景关键词**：加薪 / 销售 / 约会 / 育儿
> **核心逻辑**：AIDA (Attention, Interest, Desire, Action) + 价值重构

### 模板 A：职场加薪/晋升
1.  **Attention (感性铺垫)**：
    *   “老板，最近项目这么紧，其实团队每个人压力都很大（看见情绪），特别是上次那个突发状况...”
2.  **Interest (打破预期)**：
    *   “但我今天不是来诉苦的。相反，我觉得这次危机其实是个转机（重构：危机=机会）。”
3.  **Desire (价值重构/故事三角)**：
    *   “在这个过程中，我发现如果不解决XX问题（迫害者），我们的效率永远上不去。所以我主动承担了YY责任（拯救者），结果不仅搞定了，还沉淀了一套SOP（互惠：给公司的资产）。”
4.  **Action (明确请求)**：
    *   “基于我对公司未来的贡献预期（预期管理），我希望薪资能调整到ZZ。这不仅是对我的激励，也是给团队的一个信号（社会认同：努力有回报）。”

### 模板 B：销售/带货
1.  **Attention (情绪标签)**：
    *   “我知道大家现在都不敢乱花钱（情绪：焦虑），觉得买什么都是智商税。”
2.  **Interest (冲突降维)**：
    *   “但咱们不是为了省钱而省钱，是为了把钱花在刀刃上（降维：不是不买，是买对的）。”
3.  **Desire (稀缺+社会认同)**：
    *   “这款产品，很多像您这样的精明买家都在抢（社会认同）。因为原材料涨价，这可能是最后一批维持原价的货了（稀缺）。”
4.  **Action (承诺一致)**：
    *   “您刚才也说了，想给孩子最好的（承诺）。现在下单，还有7天无理由退货（降低风险），试错成本为零。”

## 🛠️ Tool 3: Anki 记忆卡片包 (Memory Pack)

> **说明**：请将以下内容导入 Anki，每天复习，直至形成肌肉记忆。

**[正面]**：如果你想让人做一件事，不要告诉他“怎么做”，要描绘...？
**[背面]**：**“做完之后的美好图景”**（画饼艺术/Inception）。
**[场景]**：卖音响不要讲参数，要讲“红酒、沙发、大提琴震动地板”。

**[正面]**：处理冲突的第一步是什么？
**[背面]**：**“情绪防爆拆除”**。
**[口诀]**：看见 -> 确认 -> 询问需求。先处理心情，再处理事情。

**[正面]**：西奥迪尼六大原理中，哪一个利用了“亏欠感”？
**[背面]**：**互惠 (Reciprocity)**。
**[实战]**：先给个小恩惠（可乐/试用装），对方就会觉得必须回报你（买彩票/下单）。

**[正面]**：黄执中的“故事三角”包含哪三个角色？
**[背面]**：**受害者、迫害者、拯救者**。
**[应用]**：别把自己当受害者（抱怨），要把对方当拯救者（邀请合作）。

**[正面]**：当谈判陷入僵局（立场对立）时，应该怎么做？
**[背面]**：**“冲突降维”**。
**[方法]**：剥离立场（我要开窗），挖掘需求（我要空气），寻找第三条路（开隔壁窗）。

**[正面]**：什么叫“拒绝-后撤”策略？
**[背面]**：**先提大要求，被拒后再提小要求**。
**[原理]**：利用“互惠”原理。你的让步（从大到小）会被视为恩惠，对方觉得他也得让步（同意小要求）。

**[正面]**：如何利用“承诺一致”原理搞定客户？
**[背面]**：**“登门槛”效应**。
**[方法]**：先诱导他答应一个小请求（贴个标签/试用），为了维持形象一致，他很难拒绝后续的大请求（买产品）。

**[正面]**：为什么“丑话说在前头”反而能提高满意度？
**[背面]**：**预期管理**。
**[公式]**：满意度 = 体验 - 预期。降低预期，哪怕体验一般，满意度也会提升。
