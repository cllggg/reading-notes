# 02. 人类误判心理学：修复认知固件的25个Bug

> **“我们的大脑就像一个有缺陷的遗留系统（Legacy System），虽然经过了数百万年的进化，但很多代码还是针对原始丛林环境编写的。”**

如果把人脑看作一套操作系统，那么芒格总结的“人类误判心理学”就是一份详尽的**Bug 报告**。马斯克和黄仁勋之所以能做出反直觉的决策，是因为他们不仅意识到了这些 Bug，还学会了如何利用它们（黑客思维）。

以下是几个最致命的 Bug，以及顶级创业者是如何修复或利用它们的。

## Bug #1: 避免不一致性倾向 (Inconsistency-Avoidance Tendency)

*   **症状**：大脑像一个为了节省内存而拒绝更新缓存的浏览器。一旦我们做出了某个决定或养成了某个习惯，就极难改变。
*   **企业版症状**：诺基亚拒绝触屏，柯达拒绝数码相机。
*   **马斯克的 Patch（补丁）**：**第一性原理重置**。
    *   他不在乎“汽车一直是这样造的”，他只在乎“物理学允许汽车怎么造”。
    *   如果你发现昨天的决定是错的，今天就立刻改过来，哪怕要拆掉整个生产线。**面子是这个世界上最不值钱的东西。**

## Bug #2: 社会认同倾向 (Social-Proof Tendency)

*   **症状**：羊群效应。当情况不确定时，我们倾向于看别人怎么做。
*   **黄仁勋的 Patch**：**孤独的远见**。
    *   当所有芯片厂商都在抢占 CPU 市场时，英伟达花了十年时间打磨当时没人用的 CUDA。
    *   **黄仁勋心法**：“如果你想做出独一无二的东西，你必须习惯被误解，必须习惯长期孤独。” 不要看竞争对手在做什么，看未来的物理规律指向哪里。

## Bug #3: 奖励和惩罚超级反应倾向 (Reward and Punishment Superresponse Tendency)

*   **症状**：人们对激励机制的反应大到不可思议。你奖励什么，你就得到什么。
*   **马斯克的 Exploit（利用）**：**极端的期权激励**。
    *   马斯克给特斯拉设定的薪酬计划被认为是“不可能完成的任务”（市值从500亿涨到6500亿）。但他知道，只要奖励足够疯狂，人类的主观能动性就能被无限激发。
    *   **警惕**：如果激励机制设计得不好（例如只奖励代码行数），你会得到一堆垃圾代码。

## Bug #4: 简单的、避免痛苦的心理否认 (Simple, Pain-Avoiding Psychological Denial)

*   **症状**：现实太残酷，所以假装没看见。
*   **创业者版症状**：无视糟糕的财报，无视用户的差评，假装产品很完美。
*   **硅谷法则**：**直面残酷的现实（Confront the Brutal Facts）**。
    *   马斯克在 Model 3 产能地狱期间，睡在工厂地板上，盯着每一个焊点。他没有否认问题，而是把自己扔进问题的中心。
    *   **芒格金句**：“未雨绸缪总比亡羊补牢好，但在必须亡羊补牢时，你最好动作快点。”

## Bug #5: 废话倾向 (Twaddle Tendency)

*   **症状**：聪明人最大的浪费就是花时间听蠢人说废话。
*   **马斯克的 Patch**：**反会议文化**。
    *   “如果会议没价值，立刻离场。这不粗鲁，浪费别人的时间才粗鲁。”
    *   “不要用缩写词，不要用行话，说人话。”

## Bug #6: 权威-错误影响倾向 (Authority-Misinfluence Tendency)

*   **症状**：因为他是专家/老板/名人，所以他是对的。
*   **黄仁勋的 Patch**：**扁平化与知识诚实**。
    *   英伟达拥有独特的扁平化结构，信息在组织内部高速流动。
    *   在真理面前，头衔一文不值。如果一个实习生的物理计算是对的，而首席科学家的直觉是错的，那么听实习生的。

## Bug #7: 妒忌/艳羡倾向 (Envy/Jealousy Tendency)

*   **症状**：看到别人成功比自己失败还难受。
*   **芒格的解药**：妒忌是唯一一种你无法从中获得任何乐趣的罪恶。
*   **极客的解药**：**关注使命，而非竞争**。
    *   马斯克公开了特斯拉的专利。为什么？因为他的目标不是打败福特，而是加速世界向可持续能源的转变。如果别人能造出更好的电动车，那也是人类的胜利。

---

**给系统管理员（你）的建议：**

1.  **打印这份 Bug 列表**：贴在你的工位上。
2.  **每日扫描**：在做重大决策前，过一遍清单。“我现在的决定是因为‘社会认同’吗？”“我是否因为‘避免痛苦’而拒绝承认失败？”
3.  **系统升级**：当你能熟练识别并规避这些 Bug 时，你就完成了从 Windows 95 到 Linux 内核的升级。
